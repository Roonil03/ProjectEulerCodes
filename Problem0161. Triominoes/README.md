# Triominoes
Problem 161
### Description:
A triomino is a shape consisting of three squares joined via the edges.
There are two basic forms:

<p class="center"><img src="https://projecteuler.net/resources/images/0161_trio1.gif?1678992055" class="dark_img" alt="">

If all possible orientations are taken into account there are six:

<p class="center"><img src="https://projecteuler.net/resources/images/0161_trio3.gif?1678992055" class="dark_img" alt="">

Any $n$ by $m$ grid for which $n \times m$ is divisible by $3$ can be tiled with triominoes.<br>
If we consider tilings that can be obtained by reflection or rotation from another tiling as different there are $41$ ways a $2$ by $9$ grid can be tiled with triominoes:

<p class="center"><img src="https://projecteuler.net/resources/images/0161_k9.gif?1678992055" class="dark_img" alt="">

In how many ways can a $9$ by $12$ grid be tiled in this way by triominoes?

```
20574308184277971
```
